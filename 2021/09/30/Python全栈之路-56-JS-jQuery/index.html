jQuery

jQuery内部封装了原生js代码，还额外添加了很多的功能。能够让你通过书写更少的代码完成js操作，类似于python里面的模块，但是在前端不叫模块叫类库，并且jQuery的书写兼容多个浏览器，这样就不需要考虑浏览器兼容问题了。
