解决浮动带来的影响，推导步骤。

  1. 自己加一个div，设置高度。
  2. 利用clear属性
  3. 通用解决浮动带来的影响（写html之前先提前写好处理浮动带来影响的css代码，只要标签出现塌陷问题，就给响应标签加clearfix）

通用的解决方式，到哪里都是一样的，也是最正规处理塌陷问题的方法。

  1. 静态
  2. 相对定位
  3. 绝对定位

静态

所有的标签默认都是静态的，无法改变位置。

相对定位

相对于标签原来的位置做移动 relative。

绝对定位

相对于已经定位过的父标签做移动，如果没有父标签，那么就以body做参照。

固定定位

相对于浏览器窗口固定在某个位置

变量

在js中首次定义一个变量名的时候，需要用关键字声明

  • var (全局生效)
  • let (局部生效)

命名规范

JS中推荐使用驼峰命名,变量名不能使用关键字作为命名,以下是可以使用的关键字规范.

  • 数字
  • 字符
  • 下划线
  • $
