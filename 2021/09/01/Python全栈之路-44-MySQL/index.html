创建表的完整语法

注意事项

  1. 在同一张表中字段名不能重复；
  2. 字段名和字段类型是必须填写；
  3. 宽度和约束条件非必须填写;
  4. 在同一个字段名中可写多个约束条件；
  5. 最后一行字段不能加,;
类型宽度

一般情况下指的是对存储数据的限制，默认类型宽度值为’1’。只有整型括号里面的数字不是表示限制位数而是显示长度。

5.6及以下版本默认是没有开启严格模式，如果插入超过宽度的字符，程序会自动帮你截取对应宽度的字符。

5.7及以上版本默认是开启严格模式，如果插入超过宽度的字符，程序会直接给你返回报错，不会在自动截取。
